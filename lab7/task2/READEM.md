#  Library Management System

This is a simple Python project that simulates a library system.  
It shows how object-oriented programming (OOP) works.

## Features

- Books and magazines
- Borrow and return items
- Check item status
- Polymorphism demonstration

## OOP Concepts

- Abstraction (Item class)
- Inheritance (Book, Magazine)
- Encapsulation (private attributes)
- Polymorphism (different get_info methods)

## Files

- models.py → classes (Item, Book, Magazine)
- main.py → main program

## How to run

1. Open terminal  
2. Go to project folder  
3. Run:

python main.py
